5D Tools

Course Description

This course is an introduction to the languages of website making.
We will explore the basic concepts and syntax of HTML and CSS and how they work togther.

This class is the Tools component of the 5D Experience/Drawing ARTG-2224. It aims to provide you with the skills and knowledge necessary to execute your goals for projects in that course.

You should leave the class with a strong understanding of how to create a basic website with HTML and CSS, the syntax and form of these languages, and how to use online resources to learn more.

Components

  • HTML
  • CSS
  • Debugging
  • Javascript (just a little)
  • Local Host
  • Publishing Online

  • Assignment-4

    • PROJECT I BRIEF Produce a 3+ page informational website about a topic of your choice. It may be a personal portfolio or for an organization, cause, event, business, etc that you are interested in. My hope is for you to produce a product that you can host on the web and may continue to develop beyond the scope of this class. REQUIREMENTS Your project should demonstrate an understanding of meaningful, semantic HTML to create a structured web page and CSS to control typography, color, and layout. I would like you to explore, at minimum, the following HTML elements: Paragraphs, Headings, Lists, Links, Images, and Comments. And CSS: Classes, Typography, Colors, Backgrounds, and the Box Model. Check out Resources for articles and tutorials to get you started. PROJECT I IS DUE 11/7 AT THE BEGINNING OF CLASS FINAL PROJECT Simply turn the website portion of your 5D Foundations final in to me when your turn in your 5D Foundation one. Your grade for the final in this class will be calculated from a technical, coding perspective and how well you exhibit your understanding of HTML, CSS, and their relationships.
